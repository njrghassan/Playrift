# PlayRift

Live beta at: https://playrift.ghassan-njr.com/

PlayRift is a web application that analyzes player behavior to recommend games and enable profile comparison between users.

## What it does
- Analyzes user preferences and gameplay patterns
- Recommends games based on behavioral similarity
- Allows users to compare profiles and discover new games
